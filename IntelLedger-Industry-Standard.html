Technology Stack

Frontend

  • Next.js 14 + React 18
  • Tailwind CSS

  • TanStack Query

Backend

  • NestJS (Device API)
  • FastAPI (ML Service)
  • Node.js (Blockchain)

  • TimescaleDB

  • Apache Kafka

IoT Edge

  • AWS IoT Core
  • Greengrass
  • MQTT 5.0

ML/AI

  • AWS SageMaker

  • TorchServe
  • MLflow

Blockchain

  • Polygon PoS
  • Hardhat
  • The Graph
  • Solidity

Key Features

❌ Disadvantages
